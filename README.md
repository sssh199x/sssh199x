## Java Web Development Curriculum

I developed a progressive, hands-on curriculum for my CS5054 Advanced Programming Techniques module, deployed via GitHub Classroom under [**icp-year-2**](https://github.com/icp-year-2).

```
Week 1-2  →  Terminal CLI → JDBC/MySQL
Week 3-4  →  Static HTML/CSS → JSP/Servlets (MVC)
Week 5-6  →  BCrypt Security → JSTL/EL
Week 7    →  Session Management & Auth Filters
Week 8-9  →  JSP Templating → File Upload Handling
Week 10   →  Admin Dashboard Analytics
```

Each week has a **tutorial** and **workshop** repo with numbered TODO stubs, inline hints, solution branches, architecture diagrams, and test cases.
